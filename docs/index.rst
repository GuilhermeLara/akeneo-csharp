Akeneo.NET's documentation
==========================

Akeneo.NET is a .NET client for `Akeneo PIM <https://www.akeneo.com/>`_'s `RESTful API <https://api.akeneo.com/>`_. It can be run on .NET Core as well as .NET Framework. The source code to the project can be found on `Github <https://github.com/pardahlman/akeneo-csharp>`_.
